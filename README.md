# Sistema-confort
Sistema de control y actuación de confort para una habitación mediante arduino
# Componentes
- Arduino uno
-	Sensor DHT11
-	LED RGB
•	3 resistencias de 220Ω
-	Pulsador
•	1 resistencia de 10kΩ
-	Sensor de presencia PIR
-	Pantalla LCD
-	Sensor de luminosidad LDR
•	1 resistencia de 10kΩ
